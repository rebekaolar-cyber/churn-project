# Customer Churn Prediction Model 📉

End-to-end ML pipeline to predict customer churn from behavioral
and transactional data — enabling proactive retention before a
customer is already gone.

## Problem
Businesses lose revenue to churn they could have predicted.
This model identifies at-risk customers early so retention
efforts can be targeted rather than scattered.

## What's in here
- `eda.ipynb` — exploratory data analysis, distribution checks, correlation heatmaps
- `model.ipynb` — feature engineering, train/test split, model training & evaluation
- `requirements.txt` — dependencies

## Tech stack
- Python · pandas · NumPy · scikit-learn · Matplotlib · Seaborn
